 🚍 **Smart School Bus Attendance & Safety Monitoring System**

📌**Overview**

The Smart School Bus Attendance & Safety Monitoring System is a cloud-based, real-time platform designed to enhance the safety and efficiency of school transportation. It automates student attendance using QR codes and provides live GPS tracking of school buses.



🚀 **Features**
- QR Code-based attendance system  
- Real-time GPS tracking  
- Push notifications & email alerts  
- Admin dashboard for management  
- Attendance history and analytics  
- Secure role-based authentication  



🏗️ **System Modules**
- **Admin Web Portal** – Manage students, buses, and routes  
- **Driver Mobile App** – Scan QR codes and update GPS  
- **Parent Mobile App** – Track bus and receive notifications  



🛠️**Tech Stack**
- Frontend: React.js, React Native (Expo)  
- Backend: Spring Boot (Java)  
- Database: Firebase Firestore  
- Authentication: JWT  
- Notifications: Firebase Cloud Messaging, Gmail SMTP  
- Maps: Google Maps API  



⚙️**Working Flow**
1. Each student gets a unique QR code  
2. Driver scans QR while boarding/dropping  
3. Attendance is recorded in database  
4. GPS updates bus location in real-time  
5. Parents receive instant notifications
